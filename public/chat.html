Tài khoản đang quản lý

Mỗi tài khoản đi kèm một proxy riêng để tránh Zalo phát hiện đa-tài-khoản cùng IP.

Định dạng Proxy

Hỗ trợ các dạng sau (paste trực tiếp vào ô Proxy):
  • host:port — proxy public không auth (vd: 1.2.3.4:8080)
  • host:port:user:pass — proxy có auth (vd: sp22v2-vn01.top1proxy.com:15140:v2-140-895175:TDEVJ)
  • http://user:pass@host:port — định dạng URL chuẩn
  • socks5://user:pass@host:port — proxy SOCKS5

Cập nhật Page Access Token

Trang: —
Page ID: —

Cách lấy Page Access Token đúng quyền:
  1. Vào Graph API Explorer
  2. Chọn Page (không phải User) → chọn Fanpage cần lấy token
  3. Tick các quyền (Permissions):
    • pages_messaging ⭐ bắt buộc
    • pages_read_engagement
    • pages_read_user_content
    • pages_manage_metadata
  4. Bấm Generate Access Token → đồng ý quyền
  5. Copy token → paste bên dưới
